# Social Spark App

## 📌 Purpose of the App
The Social Spark App is a simple Android application designed to help users stay socially connected throughout the day.  
It provides small social interaction suggestions based on the time of day entered by the user.

This app was created as part of an assignment for learning basic Android development using Kotlin.

---

## 🧠 How the App Works
1. The user types a time of day into the input field (e.g. "Morning", "Afternoon", "Dinner").
2. The app processes the input using decision-making logic (if/else and when statements).
3. A social suggestion is displayed based on the input.
4. The user can reset the app using the reset button.

---

## ⚙️ Features
- Text input for time of day
- Suggestion output based on user input
- Reset button to clear input and result
- Basic error handling for invalid or empty input
- Simple and user-friendly interface

---
